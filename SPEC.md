# 银行信贷业务授信项目管理平台规范

## 1. 项目概述

- **项目名称**: CreditManagementPlatform (信贷项目管理平台)
- **项目类型**: 单页Web应用 (SPA)
- **核心功能**: 跟踪银行信贷项目的贷前调查、贷中审批、贷后管理全流程进度
- **目标用户**: 银行信贷业务人员、风险管理人员、审批人员

## 2. UI/UX 规范

### 布局结构
- **顶部导航栏**: 固定高度60px，包含Logo、导航菜单、用户信息
- **侧边栏**: 宽度240px，可折叠，包含主要功能菜单
- **主内容区**: 自适应宽度，最小宽度800px
- **响应式断点**:
  - 桌面端: > 1200px (完整布局)
  - 平板端: 768px - 1200px (侧边栏折叠)
  - 移动端: < 768px (底部导航)

### 视觉设计
- **主色调**:
  - 主色: #1a365d (深蓝色，体现银行专业性)
  - 次色: #2c5282 (中蓝色)
  - 强调色: #38a169 (绿色，用于成功状态)
  - 警告色: #dd6b20 (橙色)
  - 危险色: #e53e3e (红色)
- **背景色**:
  - 页面背景: #f7fafc (浅灰白)
  - 卡片背景: #ffffff
  - 侧边栏背景: #1a202c (深色)
- **文字色**:
  - 主文字: #2d3748
  - 次文字: #718096
  - 浅文字: #a0aec0
- **间距系统**: 4px 基础单位 (4, 8, 12, 16, 24, 32, 48, 64)
- **圆角**:
  - 小元素: 4px
  - 卡片: 8px
  - 模态框: 12px
- **阴影**:
  - 卡片阴影: 0 1px 3px rgba(0,0,0,0.12)
  - 悬浮阴影: 0 4px 12px rgba(0,0,0,0.15)
  - 模态框阴影: 0 10px 40px rgba(0,0,0,0.2)

### 字体
- **主字体**: "Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif
- **等宽字体**: "Source Code Pro", monospace (用于金额显示)
- **字号**:
  - 大标题: 28px
  - 标题: 20px
  - 副标题: 16px
  - 正文: 14px
  - 小字: 12px

### 组件
- **项目卡片**: 显示项目名称、客户、金额、当前阶段、进度条
- **流程进度条**: 三阶段可视化 (贷前/贷中/贷后)
- **数据表格**: 分页、排序、筛选
- **表单组件**: 输入框、下拉框、日期选择器
- **状态标签**: 待处理/进行中/已完成/已拒绝
- **统计卡片**: 显示关键指标 (项目数、放款额、通过率等)

### 动画
- **过渡时长**: 0.2s (快速), 0.3s (标准), 0.5s (慢速)
- **缓动函数**: ease-out
- **页面切换**: 淡入淡出 + 轻微上移
- **进度条**: 动画填充

## 3. 功能规范

### 3.1 项目管理
- **项目列表**:
  - 显示所有信贷项目
  - 支持按状态、阶段、金额范围筛选
  - 支持搜索 (项目名称、客户名称)
  - 支持排序 (创建时间、金额、状态)
- **项目详情**:
  - 项目基本信息 (项目编号、名称、类型、金额)
  - 客户信息 (企业名称/个人姓名、联系方式)
  - 业务流程进度 (贷前/贷中/贷后各阶段)
  - 附件资料
  - 审批历史记录
- **新建项目**:
  - 填写项目基本信息
  - 填写客户信息
  - 设置授信方案 (金额、期限、利率)
  - 上传资料

### 3.2 流程阶段管理
- **贷前调查阶段**:
  - 任务: 客户尽职调查、资料收集、信用评估、实地考察
  - 状态: 待开始/进行中/已完成
- **贷中审批阶段**:
  - 任务: 初审、复审、终审、合同签订
  - 状态: 待开始/进行中/已完成
- **贷后管理阶段**:
  - 任务: 首次检查、定期检查、风险预警、到期处理
  - 状态: 待开始/进行中/已完成

### 3.3 数据统计
- **概览仪表板**:
  - 总项目数 (本年/本月)
  - 待处理任务数
  - 审批中项目数
  - 已放款总额
  - 通过率统计
- **趋势图表**:
  - 月度项目数量趋势
  - 放款金额趋势

### 3.4 数据存储
- 使用浏览器 localStorage 存储项目数据
- 数据结构:
  ```json
  {
    "projects": [...],
    "customers": [...],
    "tasks": [...],
    "settings": {...}
  }
  ```
